This package includes:
→ 16 textures (Albedo, AO, Metallic Smoothness and Normal Map), size 1024x1024
→ 1 model with 20 models inside, with 87,087 in total
→ 7 prefabs
→ 1 scene with all the prefabs
→ 7 materials, for the character, and another for a skybox effect on scene

This model contains:
→ 1 body and face features (eyes, pupils, iris, eyebrowns, eye glow)
→ 4 different shapes of face
→ 1 shirt
→ 1 hoodie
→ 1 shorts
→ 1 pants
→ 1 pair of boots
→ 1 pair of sneakers
→ 3 different hairs
→ 5 color variation for each cloth

The model supports Blendshapes. In order to check them, select the gameobject "MDL_Char_WomanBody" inside the prefab "PFB_Char_CartoonWoman".
At the Hierachy tab, you'll find the Blendshapes options.

In order to change the character's clothes, open "PFB_Char_CartoonWoman" and navigate between the clothe's gameobjects. Set active the clothes you want.
The hairs can be found inside Head's bone at the Armature.
